# Agent Commands

Agents are discoverable identities exposed by Spiderweb over the unified-v2 control plane.

## agent list

List available agents.

**Examples:**
```bash
ziggystarspider agent list
```

## agent info <agent_id>

Show details for one agent.

**Arguments:**
- `agent_id` - Agent identifier (for example `mother`)

**Examples:**
```bash
ziggystarspider agent info mother
ziggystarspider agent get mother
```
